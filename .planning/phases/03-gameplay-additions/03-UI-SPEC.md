| Property | Value |

## Component Contracts

### ActionBubble (new ‚Äî `client/src/components/ActionBubble.tsx`)

A single animated pill displayed over a seat.

**Structure:** Pill-shaped span with Neon Strip styling matching the `Badge` primitive recipe.

**Visual anatomy:**
```
‚îå‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îê
‚îÇ  [FOLD]  or  [CALL 100]  or ... ‚îÇ  ‚Üê uppercase action verb + optional amount
‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îò
  border: 1.5px solid <action-color at 50% alpha>
  background: rgba(10,10,14,0.88) + backdrop-blur(12px)
  box-shadow: 0 0 8px <glow-token>, inset 0 0 4px <glow-token>
  border-radius: 999px (pill)
  padding: 4px 10px
  font: 12px / 700 / uppercase / letter-spacing 0.05em
  color: <action-color CSS var>
  text-shadow: 0 0 6px <glow-token>
  white-space: nowrap
```

**Action ‚Üí tier mapping (from `tokens.ts` ActionTier):**
| Action payload value | Tier | Color token | Glow token |
|----------------------|------|-------------|------------|
| `fold` | `fold` | `--color-action-fold` | `--glow-fold` |
| `check` | `call` | `--color-action-call` | `--glow-call` |
| `call` | `call` | `--color-action-call` | `--glow-call` |
| `bet` | `raise` | `--color-action-raise` | `--glow-raise` |
| `raise` | `raise` | `--color-action-raise` | `--glow-raise` |
| `allIn` | `allin` | `--color-action-allin` | `--glow-allin` |

**Label copy (exact strings):**
| action | amount present | Label |
|--------|----------------|-------|
| fold | ‚Äî | `FOLD` |
| check | ‚Äî | `CHECK` |
| call | yes | `CALL {amount}` |
| bet | yes | `BET {amount}` |
| raise | yes | `RAISE TO {amount}` |
| allIn | yes | `ALL-IN {amount}` |
| allIn | no (0) | `ALL-IN` |

Amount is the `amount` field from `ActionBubbleEvent` payload. Format: integer, no thousands separators (chips values at these stakes are ‚â§ 5 digits).

**Animation ‚Äî `motion/react` `AnimatePresence`:**

Normal motion:
- Enter: `initial={{ scale: 0.8, opacity: 0 }}` ‚Üí `animate={{ scale: 1, opacity: 1 }}` ‚Äî duration 120 ms, ease `easeOut`.
- Hold: 900 ms minimum (D-04) ‚Äî enforced by `useEffect` timeout before dequeue.
- Exit: `exit={{ opacity: 0, y: -6 }}` ‚Äî duration 200 ms, ease `easeIn`.

`prefers-reduced-motion` (D-06): detect via `window.matchMedia('(prefers-reduced-motion: reduce)')`. If true:
- Enter: `initial={{ opacity: 0 }}` ‚Üí `animate={{ opacity: 1 }}` ‚Äî duration 0 ms (instant snap, no scale, no drift).
- Exit: `exit={{ opacity: 0 }}` ‚Äî duration 0 ms.
- Hold: 900 ms unchanged ‚Äî bubble is still displayed, action is still readable.

### ActionBubbleLayer (new ‚Äî `client/src/components/ActionBubbleLayer.tsx`)

Mounts once inside `GameRoom`. Absolutely positioned layer covering the entire table area at `z-index: 30`.

**Positioning:** Each bubble is positioned using the same `SEAT_POSITIONS_DESKTOP` / `SEAT_POSITIONS_MOBILE` arrays already in `SeatsDisplay.tsx`. The bubble anchors to `{ left: pos.left, top: pos.top }` with the same `transform: pos.align` translate, then applies an additional upward offset:
- Standard seats: `translateY(-100%) translateY(-8px)` ‚Äî places bubble above the seat card top edge, 8px gap.
- "My seat" mobile expanded layout (seat index == mySeat, isMobile): `translateY(-100%) translateY(-48px)` ‚Äî accounts for the large card area rendered above the info strip.

Rotation offset (viewer at bottom) is applied identically to `SeatsDisplay`'s `visualIndex = (i - rotationOffset + 6) % 6` computation.

**State shape:** `Map<seatIndex, ActionBubble[]>` where each queue is a FIFO array. Per-seat: the head bubble renders; when it expires (900 ms hold + exit animation ~200 ms = 1100 ms total before dequeue), the next in queue renders. Queues are independent ‚Äî no global serialization.

**Pointer events:** `pointerEvents: 'none'` on the layer. Bubbles must not intercept touch/click events on seats.

### HandHistoryList (new ‚Äî `client/src/components/HandHistoryList.tsx`)

Rendered inside the Profile ‚Üí History tab, replacing the Phase 2 stub.

**Container:** `display: flex; flex-direction: column; gap: 8px` ‚Äî 8px between rows. No horizontal padding (Card primitive provides 16px padding on the outer wrapper).

**Loading state:** While socket request is in-flight, render a neutral Card with centered `--color-neutral` text: `"Loading hand history..."` (13px, weight 400).

**Empty state:** Neutral Card with vertical stack (gap 14px, center-aligned):
- Icon area: 56px circle, `1.5px dashed` neutral border at 50% alpha, `‚ôÝ` glyph at 28px in `--color-neutral` ‚Äî replaces the Phase 2 timer emoji (same size/position contract from Phase 2 stub preserved).
- Heading: `"No hands yet"` ‚Äî 14px weight 600, white.
- Body: `"Your played hands will appear here."` ‚Äî 13px weight 400, `--color-neutral`, `line-height: 1.5`, `max-width: 280px`.

**Error state:** Fold-tier Card with centered text: `"Could not load hand history."` followed by `"Try closing and reopening your profile."` on a second line ‚Äî 13px, `--color-action-fold`.

### HandHistoryRow (new ‚Äî `client/src/components/HandHistoryRow.tsx`)

One Card primitive per hand, initially collapsed. Tap to expand.

**Collapsed state:**
```

‚îÇ  [date label]     [table name]     [net delta]   ‚îÇ
‚îÇ  11px neutral     11px neutral     13px mono      ‚îÇ
‚îÇ  "3h ago"         "Standard 10/20"  "+250" green  ‚îÇ
‚îÇ                                    "-100" red      ‚îÇ
‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îò
```
- Card variant: `neutral` (collapsed), `active` (expanded ‚Äî cyan border).
- Card padding: 12px.
- Row layout: `display: flex; justify-content: space-between; align-items: center`.
- Date: relative time string (e.g. `"3h ago"`, `"2m ago"`, `"yesterday"`) ‚Äî 11px, weight 700, `--color-neutral`.
- Table name: 11px, weight 700, `--color-neutral`. Truncated with `overflow: hidden; text-overflow: ellipsis; max-width: 100px`.
- Net delta: 13px, weight 800, monospace, `fontVariantNumeric: 'tabular-nums'`. Positive ‚Üí `--color-action-sit` with `text-shadow: 0 0 6px var(--glow-sit)`. Negative ‚Üí `--color-action-fold` with `text-shadow: 0 0 6px var(--glow-fold)`. Prefix: `+` for wins (always explicit), `-` for losses.
- Full tap target on the row card (`cursor: pointer`, `active:scale-[0.99]`).

**Expanded state (tap to toggle):**
Reveals below the collapsed row content, separated by a `1px solid rgba(176,190,197,0.12)` divider:
```
  Board:  [5 board cards rendered as mini Card images, scale 0.55]
  Your cards: [2 hole cards, scale 0.55]  (only own cards ‚Äî always shown)
  Opponents' cards: [shown only when showedDown=true ‚Äî privacy rule D-18]
```
- Board and hole cards use the existing `HandDisplay` component with `size` prop set so cards render at approximately 32px wide each.
- Label above board: `"BOARD"` ‚Äî 9px, weight 800, `--color-neutral`, uppercase, letter-spacing 0.08em.
- Label above hole cards: `"YOUR CARDS"` ‚Äî same style as BOARD label, `--color-action-call` color for own cards.
- Opponents' cards section rendered only when `showedDown === true` for that row. Label: `"SHOWN AT SHOWDOWN"` ‚Äî 9px, weight 800, `--color-neutral`.
- Expand/collapse animation: `motion/react` height animation `initial={{ height: 0, opacity: 0 }}` ‚Üí `animate={{ height: 'auto', opacity: 1 }}` ‚Äî 180 ms, easeOut. Honors `prefers-reduced-motion` (instant if reduced).
- Gap between board/cards sections: 12px.

**Result badge:** Pill Badge to the right of date ‚Äî `variant="sit"` for win (`WIN`), `variant="fold"` for loss (`LOST`), `variant="neutral"` for chop (`CHOP`).

## Interaction States

### ActionBubble lifecycle

```
Event received
  ‚îî‚îÄ Enqueue to per-seat queue (Map<seat, Bubble[]>)
      ‚îî‚îÄ If seat queue was empty: immediately render head bubble
          ‚îî‚îÄ After 900ms hold + 200ms exit: dequeue, render next (if any)
      ‚îî‚îÄ If seat queue was non-empty: wait for head to expire, then render
```

Touch / reduced-motion:
- `prefers-reduced-motion: reduce` ‚Üí no scale, no y-drift, instant in/out. Hold unchanged at 900 ms.
- No tap interaction on bubbles (pointer-events: none on layer).

### HandHistoryRow expand/collapse

- Tap anywhere on collapsed row Card ‚Üí expand (Card border switches to `active`/cyan variant).
- Tap anywhere on expanded row Card ‚Üí collapse.
- Only one row can be expanded at a time ‚Äî expanding a new row collapses the previously expanded row.
- Expansion state is local to the `HandHistoryList` component (single `expandedHandId: string | null` state).

### History tab data loading

- Tab becomes active ‚Üí `socket.emit('getHandHistory')` fires immediately.
- Response `handHistoryData` event ‚Üí renders list (or empty state if `rows.length === 0`).
- Socket error or no response within 5 seconds ‚Üí renders error state.
- No pull-to-refresh in Phase 3. Reload on tab re-enter (socket emit fires again when tab is re-selected).

## Accessibility

- ActionBubbleLayer: `aria-live="polite"` region so screen readers announce bubbles. Each bubble has `role="status"` with text content matching the label.
- HandHistoryList: `role="list"`, each HandHistoryRow is `role="listitem"`.
- HandHistoryRow expand button: `aria-expanded={isExpanded}`, `aria-label="Hand from {date} at {table}"`.
- Board and hole card sections: `aria-label="Board cards"` / `aria-label="Your hole cards"`.
- Touch targets: entire HandHistoryRow card is the tap target (min 44px height enforced by Card + 12px padding + row content).
